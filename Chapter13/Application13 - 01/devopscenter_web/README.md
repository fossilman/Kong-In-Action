# DevopsCenter_Web

## 项目启动

```bash
$ npm install
$ npm run serve
```

## 编译

### dev 环境

```bash
# 修改.env.development 文件，修改Kong网关地址
$ npm install
$ npm run build:dev
```

### test 环境

```bash
# 修改.env.test 文件，修改Kong网关地址
$ npm install
$ npm run build:test
```

### prod 环境

```bash
# 修改.env.production 文件，修改Kong网关地址
$ npm install
$ npm run build:prod
```
